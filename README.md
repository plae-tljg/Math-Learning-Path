## Setup Guide

for setting up ubuntu  

```bash
npm install
npm run dev
```

if want to deploy to github page, please use:  

```bash
./deploy-gh-pages.sh
```

Remember that deploying will delete `node_modules`, so need do again

```bash
npm install
```
